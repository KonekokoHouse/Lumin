Lumin ClickGUI · Apple 极简风格设计稿（风格需贴合 iOS / macOS 精致感，和「Lumin」光感气质统一）

一、整体定位
- 风格：Apple 极简 + 柔光磨砂 + 无冗余元素
- 关键词：干净、留白、细线条、弱阴影、微发光、丝滑动画
- 不搞花里胡哨：无描边爆炸、无强光闪、无复杂纹理

二、配色（直接复制十六进制）
- 背景： #1A1A1A （超纯黑底，接近 macOS 深色模式）
- 面板磨砂： rgba(28, 28, 30, 0.85) 
- 分割线： rgba(255,255,255,0.1) 
- 主光色（Luminous）： #0A84FF （Apple 原生蓝）
- 次光/强调： #5E5CE6 （淡紫蓝，更优雅）
- 文字：
- 标题： #FFFFFF 
- 描述： rgba(255,255,255,0.6) 
- 禁用： rgba(255,255,255,0.3) 

三、布局结构（苹果式双栏）
主窗口
- 大圆角：20px
- 无边框，只有极弱内阴影，模拟悬浮感
- 固定宽高，不拉伸变形
  1. 顶部标题栏
- 左： Luminous （细体、居中对齐）
- 右：极简小版本号  v1.0 
- 无任何图标、无花纹
                       2. 左侧导航栏
- 宽度固定，小圆角
- 选项：Modules / HUD / Settings / Configs
- 选中：背景浅灰高亮 + 文字白色
- 未选中：文字灰色
  3. 右侧内容区
- 模块列表 + 设置项
- 大量留白，间距宽松

四、模块条目设计（核心质感）
- 高度：宽松、不拥挤
- 圆角：14px
- 背景：默认透明
- 悬浮： rgba(255,255,255,0.05)  浅高光
- 启用状态：
- 左侧一个小圆点（主色 #0A84FF，微发光）
- 文字变白、略加粗
- 点击展开设置：平滑向下展开，无跳变

五、控件样式（全苹果风）
1. Toggle 开关
- 扁长、细边框
- 关闭：灰色描边，空心
- 开启：填充  #0A84FF ，无外发光
- 动画：0.2s 平滑滑动，仿 iOS

2. Slider 滑块
- 极细轨道
- 进度：主色渐变
- 滑块：小白圆，极简
- 拖动：顺滑+轻微弹性

3. Mode 选择器
- 横向小按钮组
- 圆角统一
- 选中：浅背景+主色文字
- 无描边、无强光

4. 输入框 / 颜色选择器
- 极简圆角矩形
- 聚焦时轻微高光
- 颜色块：小圆角，无边框

六、文字与字体
- 字体：无衬线细体
- 模块名：中等字重
- 说明文字：轻量字重，灰色
- 不使用描边、不使用阴影，只靠颜色区分层级

七、动画规则（Apple 灵魂）
- 所有过渡：0.2~0.3s 平滑
- 打开 GUI：轻微缩放+淡入
- 展开设置：高度平滑动画
- 开关/滑块：线性+轻微缓动
- 无抖动、无弹跳过度、无闪屏

八、一句话 slogan（放 GUI 角落）

Lumin · Simple. Clean. Light.
